## ¿Qué es BBP LABS?

**BBP LABS** es una plataforma de práctica de bug bounty donde puedes resolver labs interactivos basados en vulnerabilidades reales reportadas en programas de [HackerOne](https://hackerone.com), [Bugcrowd](https://bugcrowd.com), [Intigriti](https://intigriti.com) y [YesWeHack](https://yeswehack.com).

Cada lab replica una vulnerabilidad real en un entorno seguro y aislado. Practica, aprende y mejora tus habilidades de hacking ético.

**LINK:** [https://bblabs.es](https://bblabs.es)

## Características

- **Labs basados en reportes reales** — Vulnerabilidades replicadas de programas reales de bug bounty
- **Nuevos labs cada semana** — Contenido actualizado con las técnicas más recientes
- **Writeups paso a paso** — Cada lab incluye una solución detallada
- **Sistema de flags (CTF)** — Resuelve labs y sube en el ranking
- **Academy gratuita** — Tips, payloads y metodologías para 16 categorías de vulnerabilidades y contextos de ataque
- **Ranking de hunters** — Compite con otros hunters por completar más labs
- **Creadores** — Hunters reales crean los labs desde su experiencia en bug bounty

## Categorías de Labs

| Categoría | Descripción |
|---|---|
| **XSS** | Cross-Site Scripting — Stored, Reflected, DOM, Blind |
| **IDOR** | Broken Access Control — Acceso a recursos de otros usuarios |
| **Auth Bypass** | JWT, 2FA, OTP — Bypasses en autenticación |
| **SSRF** | Server-Side Request Forgery — Requests a servicios internos |
| **SQLi** | SQL & NoSQL Injection — Inyección en bases de datos |
| **File Upload** | Subida de archivos maliciosos — Webshells, SVG XSS |
| **CORS** | Misconfiguration — Exfiltración de datos cross-origin |
| **Race Conditions** | Condiciones de carrera — Double-spending, cupones |
